BrainXAOMI


Recover motor function
with active-assisted system

"
BrianXAOMI is a brain-computer interface device used for arm
and hand rehabilitation in stroke patients. It is a new therapeutic strategy
that can help stroke patients with moderate to severe motor impairment
are able to move arm and hand by using only their thoughts.
"

Our product enhances

NEUROPLASTICITY,

which refers to the connections formed by axons that project from neurons
to make synapses onto neurons in another location, enabling neurotransmission
(the sending of signals from one region of the nervous system to another)
